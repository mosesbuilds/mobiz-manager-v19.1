## üñ•Ô∏è Environment & Requirements

MoBiz Manager v19.1 is compatible with **Python 3.x** and the **Tkinter GUI library**.

### Supported Platforms
- **Windows:** Fully supported  
- **Linux:** Fully supported (requires `python3-tk`)  
- **Android:** Supported via **Pydroid 3** or compatible Python IDE

### Setup Guide

**Windows / Linux**
1. Install Python 3.x from [python.org](https://www.python.org/).  
2. Ensure Tkinter is installed:  
   - Linux (Debian/Ubuntu): `sudo apt install python3-tk`  
3. Download `mobiz_manager_v19.py` and run:
   ```bash
   python3 mobiz_manager_v19.py
   ```

**Android (Pydroid 3)**
1. Install **Pydroid 3** from Google Play.  
2. Place `mobiz_manager_v19.py` in accessible storage.  
3. Open Pydroid 3 ‚Üí File ‚Üí Open ‚Üí Run the script.  
4. Ensure the app can write files for logs and license (`income_log_v19.txt`, `expense_log_v19.txt`, etc.)  

**Example Run Commands**
```bash
# Windows / Linux
python mobiz_manager_v19.py

# Android (Pydroid 3)
# Open Pydroid ‚Üí File ‚Üí Open ‚Üí Select mobiz_manager_v19.py ‚Üí Run
```
